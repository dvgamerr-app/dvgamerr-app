# Profile Resume & CV Project

> Nuxt.js project

## Build Setup

``` bash
# install dependencies
$ npm i # Or yarn i

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start
```
